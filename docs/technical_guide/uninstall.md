# Uninstall Whanos

## Keep data

To uninstall Whanos, run the following command:

```bash
sudo /opt/tools/uninstall.sh
```

Your data volumes will be preserved.

## Remove everything

To fully uninstall Whanos, run the following command:

```bash
sudo /opt/tools/uninstall.sh --remove-data
```

Your data volumes will be preserved.
